# Nonograms


### Description:
This project was inspired by Rolling Scopes School task [Nonograms.](https://github.com/rolling-scopes-school/tasks/tree/master/tasks/nonograms) The game presents to user a board, containing cells. Moreover, the board has numbers, representing the quanity of cells which must be filled. User can fill the cells and then check if their solution is correct. Game has 3 difficulty levels, differentiated by size of the board. Easy, medium and hard levels implement the board 5*5, 10*10 and 15*15 relatively.
